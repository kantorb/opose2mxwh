# OpenPose to Maxwhere Interface 

This is my project for the Information Technology of Cyberphysical Systems course at University of Technology And Economics. My neptun code starts with JZ.

## Install
1. Download [Openpose](https://github.com/CMU-Perceptual-Computing-Lab/openpose)
2. Create a `node` folder in the openpose directory (next to 3rdparty, bin, examples, etc...)
1. Run `npm i` to install dependencies.
2. Rename `mqtt-config.json.example` to `mqtt-config.json` and change settings accordingly to your setup.
